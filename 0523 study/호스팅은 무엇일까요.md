# 호스팅은 무엇일까요?

## 요약

1. 호스팅이란 서버 전체 또는 일부 공간을 임대하는 서비스를 뜻한다.

2. 호스팅엔 웹 호스팅, 서버 호스팅, 클라우드, 메일 호스팅 등 그 종류가 다양하다.

## 호스팅이란

> 호스팅(hosting)이란 서버 컴퓨터의 전체 또는 일정 공간을 이용할 수 있도록 임대해 주는 서비스를 말한다.PC 서버로 웹 사이트를 운영하려면 24시간 365일 켜야 하나 현실적으로 불가능하므로 호스팅 업체가 미리 여러 대의 서버로 이용자들에게 임대해 주고 그 대가를 받는 서비스가 호스팅이다. 호스팅 서비스에는 웹 호스팅과 서버 호스팅, 메일 호스팅 등 다양한 종류가 있다.

즉, 호스팅은 서버 컴퓨터의 전체 또는 일부 공간을 이용할 수 있도록 임대해주는 서비스이며

웹 호스팅, 서버 호스팅 등 다양한 종류가 있다.

## 웹 호스팅

여러대의 웹 사이트를 한 서버에 운영하는 방식

<li> 장점: 쉽게 관리가 가능하고, 비교적 간단하다.
<li> 단점: 오버셀링에 취약한 편이다. 한 사람이 과도한 리소스를 사용하면 다른 사람도 영향을 받는다.

## 서버 호스팅

하나의 서버 전체를 단독 사용하는 방식

<li> 장점: 서버 관리에 대한 직접적 권한을 지닌다. 서버 단독 사용으로 보안상 유리하다. 빠른 데이터 전송 속도를 누릴 수 있다.
<li> 단점: 구축 시간이 오래걸린다. 비용이 많이 든다.

## 클라우드 서버

서버 호스팅을 가상화 한 것으로, 구축한 가상 서버를 단독으로 이용할 수 있는 서비스

<li> 장점: 단독 서버의 장점과 더불어 서버 자원의 증가 감소를 유연하게 결정할 수 있음 사용한 만큼 지불하기 때문에 경제적 장점
<li> 단점: 하나의 가상 서버에 이상이 생기면 연쇄적으로 이용에 장애가 생김 → 이중화 또는 백업으로 커버 가능
